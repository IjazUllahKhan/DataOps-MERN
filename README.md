# 🚀 DataOps-MERN

### *“Search, Filter, Sort, Paginate, and Export — all in one MERN stack app.”*

---

## 📖 Overview
**DataOps-MERN** is a lightweight MERN stack project that demonstrates core data operations:  
✅ Search  
✅ Filter  
✅ Sort  
✅ Pagination  
✅ Export to CSV  

Built with **React, Node.js, Express, and MongoDB**.
